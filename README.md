Inventory Management System - README
Project Overview
Inventory Management System is a desktop-based application developed using Python and Tkinter.
The system helps businesses manage employees, suppliers, categories, products, sales, and tax
settings through a user-friendly graphical interface.
Features
• Employee Management
• Supplier Management
• Category Management
• Product Management
• Sales Tracking and Billing
• Tax Percentage Configuration
• Dashboard with Real-Time Statistics
• Secure Login and Logout System
Technologies Used
• Python
• Tkinter (GUI Development)
• MySQL Database
• PyMySQL (Database Connectivity)
Project Modules
Employee Module: Manage employee records.
Supplier Module: Store supplier information.
Category Module: Organize products into categories.
Product Module: Add, update, and manage products.
Sales Module: Generate bills and track sales.
Tax Module: Configure and save tax percentages.
Database
The application uses MySQL database 'inventory_data' to store and manage all records including
employees, suppliers, products, categories, sales, and system settings.
How to Run
1. Install Python and MySQL.
2. Install required libraries: tkinter, pymysql.
3. Create the database 'inventory_data'.
4. Update database credentials if required.
5. Run main.py to start the application.
